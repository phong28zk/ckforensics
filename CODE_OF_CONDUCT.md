# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

**Summary:**
- Be respectful. No harassment, discrimination, or personal attacks.
- Disagreements happen — keep them technical, not personal.
- Maintainers may remove comments, commits, code, issues, or PRs that violate this code.

**Reporting:** open an issue with label `coc` or email the maintainer via GitHub profile contact. Reports are handled confidentially.

Full text: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
